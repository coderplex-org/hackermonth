In this meetup, we'll introduce you to React Native, and walk you through building a simple todo app using it. Following that, we'll have a break for lunch where you can network with other developers.

From 2 pm - 6 pm, we'll have an open house pair programming session, where you can collaborate with others to build something, or go through some learning resources together.

  • You need to be familiar with ReactJS
  • Laptops are mandatory, charging ports will be available.
  • This meetup is free of cost.
